# 장부의 무게 (Ledger Weight) 프로젝트 개요

## 프로젝트 소개
- **원작**: BANG! 카드 게임
- **테마**: 상인들의 외교/정치적 이권 다툼
- **기술 스택**: FastAPI (백엔드), React Native (프론트엔드)
- **통신 방식**: WebSocket 기반 실시간 통신

## 핵심 개념

### 용어 변환
| 원작 | 장부의 무게 | 설명 |
|------|------------|------|
| HP | 재력 | 플레이어의 생명력 |
| Range | 영향력 | 카드의 공격 범위 (사거리) |
| Suit | 카드 무늬 | 검(♠), 책(♣), 치유(♥), 돈(♦) |
| Rank | 카드 숫자 | 상(A), 대(K), 중(Q), 소(J) |

### 역할 및 승리 조건
1. **상단주** (Sheriff): 상단주의 권익을 지키며 최종적으로 상권 수호
2. **원로원** (Deputy): 상단주를 보좌하며 상권 수호에 기여
3. **적도 세력** (Outlaw): 상단주를 탈핵하고 상권을 장악
4. **야망가** (Renegade): 모든 플레이어 처치 후 단독으로 상권을 독점

### 주요 카드 타입
- **공격 카드**: 정산(Bang!), 회피(Missed!), 전원 견제(Gatling) 등
- **무기/장착 카드**: 연속 상환 요구(Volcanic), 독점 상권 선언(Winchester) 등
- **특수 카드**: 비상금(Beer), 승부(Duel), 패거리 습격(Indians) 등

### 보물 시스템
- 총 16종의 장착 보물
- 각 보물은 원작 BANG! 캐릭터의 능력을 구현
- 예: 거대 금고(Willy the Kid) - 턴당 정산 무제한 사용

## WebSocket 프로토콜

### GAME_STATE_UPDATE (서버 → 클라이언트)
- 현재 턴 정보
- 모든 플레이어 상태 (재력, 영향력, 장착 보물, 핸드 카드 수)
- 덱/버림 더미 상태
- 직전 이벤트 정보

### PLAYER_ACTION (클라이언트 → 서버)
- `USE_CARD`: 카드 사용 요청
- `RESPOND_ATTACK`: 방어 카드 사용
- `END_TURN`: 턴 종료
