# 작업일지 디렉토리

이 디렉토리는 프로젝트 진행 과정을 날짜별로 기록하는 작업일지를 저장합니다.

## 구조
```
logs/
├── README.md (이 파일)
└── YYYY/
    └── MM/
        └── DD.md (날짜별 작업일지)
```

## 예시
- `logs/2025/12/11.md` - 2025년 12월 11일 작업일지

## 작업일지 작성 가이드
각 작업일지에는 다음 내용을 포함합니다:
- 작업 시간
- 완료한 작업
- 진행 중인 작업
- 이슈 및 해결
- 학습 및 참고 자료
- 다음 작업 계획
- 메모
